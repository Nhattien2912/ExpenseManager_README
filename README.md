Expense Manager
Hệ thống quản lý tài chính cá nhân đa nền tảng gồm:
Android app (Kotlin + XML)
Web dashboard (Next.js + TypeScript)
Đồng bộ dữ liệu qua Firebase Auth + Cloud Firestore
Ứng dụng giúp theo dõi chi tiêu, thu nhập, ví tiền, tiết kiệm, nợ và khoản vay. Ngoài ra còn có AI trợ lý tài chính hỗ trợ tạo giao dịch từ ngôn ngữ tự nhiên.
