# 💰 Expense Manager

A **cross-platform personal finance system** built with:

* 📱 **Android App** — Kotlin + XML
* 🌐 **Web Dashboard** — Next.js + TypeScript
* ☁️ **Cloud Sync** — Firebase Auth + Cloud Firestore

The application helps users track **income, expenses, wallets, savings, debts, and loans**.
It also integrates an **AI financial assistant** that can create transactions using **natural language input**.

# 🚀 Features

## 📱 Android

* Manage transactions: **income, expense, transfer**
* **Wallet and category management**
* **Daily / monthly financial statistics**
* **Debt and loan tracking**
* **Recurring transactions & budget alerts**
* **Android home screen widget**
* **Biometric authentication**

---

## 🌐 Web Dashboard

* Google sign-in via **Firebase Auth**
* CRUD for **transactions, categories, wallets**
* **Search and filter** transaction history
* **Monthly financial charts**

---

# 🏗 Architecture

Android application uses:

**MVVM + Repository + Room Database**

Key characteristics:

* ☁️ **Cloud synchronization** using Firebase Firestore
* 📡 **Offline-first architecture** (local data stored before syncing)
